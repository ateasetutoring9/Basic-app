The Coordinate Plane

Every point on a graph is described by two numbers written as (x, y). The x-axis runs left–right; the y-axis runs up–down. They cross at the origin (0, 0).

2 / 4 Plotting Points

How to Plot a Point

To plot (x, y): start at the origin, move x units along the x-axis, then move y units up or down.

  • (2, 3) → right 2, up 3
  • (-1, 4) → left 1, up 4
  • (3, -2) → right 3, down 2
3 / 4 Slope

Understanding Slope

Slope (written m) measures how steep a line is — how much y changes for every 1 unit increase in x.

slope (m) = rise / run = change in y / change in x
  • Positive slope → line goes up left to right
  • Negative slope → line goes down left to right
  • Zero slope → horizontal line (y never changes)

Example: the line through (0, 1) and (2, 5) has slope m = (5 - 1) / (2 - 0) = 4 / 2 = 2. For every step right, the line rises 2.

4 / 4 y = mx + b

The Slope-Intercept Form

Every straight line can be written as:

y = mx + b
  • m is the slope (steepness)
  • b is the y-intercept — where the line crosses the y-axis (when x = 0)

Example: y = 2x + 1

  • Slope = 2 (rises 2 for each step right)
  • y-intercept = 1 (crosses y-axis at y = 1)
  • To draw it: plot (0, 1), then from there go right 1 and up 2 to get (1, 3). Draw a line through both points.
